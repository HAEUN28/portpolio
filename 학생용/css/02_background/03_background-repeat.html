배경이미지 반복여부

요소의 배경 이미지를 반복하여 출력하고자 할 때는 background-repeat 속성에 repeat값을 지정하면 됩니다.
요소의 배경 이미지를 X축으로만 반복하여 출력하고자 할 때는 background-repeat 속성에 repeat-x 값을 지정하면 됩니다.
요소의 배경 이미지를 Y축으로만 반복하여 출력하고자 할 때는 background-repeat 속성에 repeat-y값을 지정하면 됩니다.
요소의 배경 이미지를 반복하여 출력하지 않고자 할 때는 background-repeat 속성에 no-repeat값을 지정하면 됩니다.
